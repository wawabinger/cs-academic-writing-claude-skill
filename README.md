# CS Academic Writing - Claude Skills 

A Claude skill for computer science academic writing, including research papers, theses, dissertations, grant proposals, and reviewer responses.

- **Complete Writing Workflow**: From initial drafting to final polishing
- **Section-Specific Guidance**: Tailored advice for Introduction, Related Work, Methodology, Results, and Conclusion
- **LaTeX Support**: Automatic detection and formatting for mathematical content
- **Style Enforcement**: Precision, concision, neutrality, and consistency checks
- **Citation Management**: Proper academic citation formats and bibliography guidance
- **Reviewer Response Templates**: Professional response patterns for peer review feedback

## üîß Installation

### Local Installation

```bash
# Clone this repository to your local machine
git clone https://github.com/wawabinger/cs-academic-writing-claude-skills.git

# Add to Claude Code skills directory
# Option A: Symlink (recommended)
ln -s /path/to/cs-academic-writing-claude-skills/skills/* ~/.claude/skills/

# Option B: Copy files
cp -r /path/to/cs-academic-writing-claude-skills/skills/* ~/.claude/skills/
```
### Via Claude Marketplace
Open claude code and input the following command in conversation
```
 /plugin marketplace add wawabinger/cs-academic-writing-claude-skill 

/plugin install cs-academic-writing@wawabinger
```
## üìä Usage

The `cs-academic-writing` skill is automatically triggered when you:

- Use phrases like "please write...", "please polish...", "help me draft...", "improve my..."
- Request help with academic content (papers, theses, proposals)
- Ask for LaTeX formatting assistance
- Need help responding to reviewer comments
